Core Features of the Platform

Fitness Data Dashboard

Personalized user dashboard that consolidates fitness data

  • Display running stats, challenges completed, and XP earned.
  • Offer insights through graphs and progress reports.

Blockchain Tokenomics

Complx Coins as the reward currency.

  • Support wallet connection (e.g., Phantom or Magic.link) to view balances, stake tokens, and redeem rewards.
  • Enable token burning and staking mechanisms for long-term value.

Community Integration

Sync Discord roles and levels with user profiles on the platform.

  • Redeem tokens for discounts, exclusive roles, or physical rewards.

E-commerce Expansion

Token-based discounts directly in the Shopify store.

  • Support payments in fiat and crypto.
  • Link purchases to user profiles, allowing for gamified shopping experiences.

Gamified Experience

A reward system tied to real-world fitness and community participation.

Transparency and Security

Blockchain integration for token management.

Secure user authentication through wallet connections.
